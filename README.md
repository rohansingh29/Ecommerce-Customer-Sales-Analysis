# Ecommerce Customer Sales Analysis

# Project Overview
This project analyzes an ecommerce dataset using PostgreSQL, Python (Pandas), and Power BI. The objective is to identify customer behavior, sales trends, product performance, payment preferences, and overall business performance through data analysis and interactive dashboards.

# Tools & Technologies
- PostgreSQL
- Python (Pandas, Numpy)
- Jupyter Notebook
- Power BI

# Dataset
The project contains five tables:
- Customers
- Orders
- OrderItems
- Products
- Payments

# Project Files
- Ecommerce_Analysis.ipynb – Data cleaning and analysis
- Ecommerce_Project.sql – SQL queries
- Ecommerce_Dashboard.pbix – Interactive Power BI dashboard
- Ecommerce Dashboard.jpg – Dashboard preview
- Customers.csv
- Orders.csv
- OrderItems.csv
- Products.csv
- Payments.csv

# Dashboard Insights
- Total Customers
- Total Orders
- Total Revenue
- Average Order Value
- Monthly Revenue Trend
- Revenue by Payment Type
- Top States by Revenue
- Top Cities by Orders
- Top Customers by Spending
- Product Category Performance

# Conclusion
This project demonstrates SQL querying, Python data analysis, data visualization, and dashboard development to generate business insights from ecommerce sales data.
